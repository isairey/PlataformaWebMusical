# 🌌 Acerca de Astro Player

**Astro Player** es una aplicación web enfocada en la exploración y reproducción de contenido musical utilizando la API de Deezer.

La plataforma permite:

- 🎵 Descubrir música
- 👨‍🎤 Buscar artistas
- 💿 Explorar álbumes
- 📂 Navegar playlists
- 🔍 Realizar búsquedas avanzadas
- 📱 Instalar como PWA

El proyecto está orientado al aprendizaje y práctica de:

- React.js
- Redux
- APIs REST
- Progressive Web Apps
- Diseño responsivo
- Arquitectura Frontend moderna

# ⚡ Instalación

## 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/isairey/PlataformaWebMusical.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaWebMusical
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar servidor de desarrollo

```bash
npm run start
```

---

## 5️⃣ Abrir en navegador

```bash
http://localhost:3000
```
